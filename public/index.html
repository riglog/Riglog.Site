RIGLOG

GitHub: https://github.com/riglog

Evidence IT vybavení, primárně serverů, desktopů a do budoucna třeba i síťových prvků atd.

Cíle projektu

  • Zjednodušení organizace IT infrastruktury. Kromě běžného seznamu zařízení, také evidence provedených prací, aktualizací, bezpečnostních problémů, restartů, evidence komponent a možná i monitoring.
  • Čistě má osobní snaha o zlepšení svých programátorských schopností, znalostí .NETu a vývoje webových, mobilních i desktopových aplikací na vlastním projektu, postaveném na zelené louce (učitě to bude běh na dlouhou trať).
  • Poznat sílu open source a pokusit se zapojit do projektu více lidí.

Struktura projektu (repozitáře)

  • Riglog.Api: Rest API backend v .NET 5
  • Riglog.App: Aplikace pro Android, iOS, možná Windows a do budoucna i MacOS (Xamarin Forms).
  • Riglog.Web: Webový frontend aplikace (Pravděpodobně Blazor, vue.js nebo něco podobného).
  • Riglog.Site: Tato jednoduchá statická informační stránka.

Stav projektu

  • Září 2021 - Přešlapování na místě. Příprava na .NET 6 a .NET MAUI
  • Červenec, Srpen 2021 - Dovolená :)
  • Červen 2021 - API: Autentizace, verzování, Swagger, základní databázový model. DevOps: Automatické nasazení API do Azure a automatické publikování docker obrazů na DockerHub.
  • Květen 2021 - Návrh struktury projektu, vytvoření informačního webu a zahájení prací na API.
